# Baltimore Therapeutic Treatment Center Project

## Overview
A responsive landing page for the Baltimore Therapeutic Treatment Center Project, built from the provided Figma design. This single-page website demonstrates modern web development practices with clean code and responsive design.

## Features
- Fully responsive design (desktop, tablet, mobile)
- Mobile-first approach
- Cross-browser compatible
- Accessible HTML structure
- Interactive elements with JavaScript
- Optimized for performance

## Technologies Used
- HTML5 (semantic markup)
- CSS3 (Flexbox, Grid, CSS Variables)
- JavaScript (ES6+)
- Font Awesome icons
- Google Fonts (Inter, Montserrat)

## Setup Instructions
1. Clone or download the project files
2. Place all files in the same directory
3. Add your images to the `assets/` folder (or update paths in CSS)
4. Open `index.html` in any modern web browser

## Responsive Breakpoints
- **Mobile**: < 768px
- **Tablet**: 768px - 992px  
- **Desktop**: > 992px

## Browser Compatibility
Tested and working on:
- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)

## Design Implementation Details
- **Colors**: Matched exact colors from Figma design
- **Typography**: Inter (body) and Montserrat (headings) as per design
- **Layout**: 1:1 implementation of Figma layout
- **Spacing**: Precise spacing and alignment

## Key Features Implemented
1. Responsive navigation with mobile hamburger menu
2. Hero section with curved bottom edges
3. Mission statement with background video/image
4. Events section with interactive cards
5. News section with alternating layouts
6. Bidding opportunities section
7. Footer with newsletter subscription

## Performance Optimizations
- Semantic HTML for better accessibility
- CSS variables for maintainability
- Optimized image references
- Minimal JavaScript for necessary interactions
- Proper asset loading
